# 🎓 LabTop: Система мониторинга лабораторных работ

![C++](https://img.shields.io/badge/C++-00599C?style=for-the-badge&logo=c%2B%2B&logoColor=white)
![Windows Forms](https://img.shields.io/badge/Windows%20Forms-512BD4?style=for-the-badge&logo=.net&logoColor=white)
![Status](https://img.shields.io/badge/Status-In%20Development-yellow?style=for-the-badge)

**LabTop** — это персональный ассистент студента, разработанный для автоматизации контроля выполнения учебных заданий. Программа избавляет от путаницы в дедлайнах и файлах, предоставляя наглядный интерфейс для отслеживания прогресса по каждой дисциплине.

---

## ✨ Основные возможности

* **📊 Централизованный учет:** Все лабораторные работы в одной таблице с подробной информацией (ID, название, дедлайн, комментарии).
* **🎨 Цветовая индикация статусов:** Программа автоматически подсвечивает задачи в зависимости от их состояния (готовность, просрочка или ожидание).
* **📂 Умная проверка файлов:** Автоматический контроль готовности работы. Система проверяет наличие необходимых артефактов в выбранной папке:
    * Отчеты
    * Блок-схемы
    * Исходный код
    * Диаграммы IDEF0
* **💾 Локальное хранение:** Все данные сохраняются в файл, что обеспечивает быстрый доступ к информации без необходимости подключения к базе данных.

## 🛠 Технологический стек

- **Язык программирования:** C++/CLI
- **Интерфейс:** Windows Forms (WinForms)
- **Контейнеры данных:** STL (std::vector)
- **Хранение данных:** Файловая система (потоковый ввод/вывод)

---

## 🚀 Интерфейс и использование

### Добавление новой работы
При нажатии на «Добавить работу» открывается форма конфигурации, где вы можете:
1. Выбрать путь к рабочей директории через удобный проводник.
2. Установить крайний срок (Deadline).
3. Выбрать чекбоксами необходимые компоненты для сдачи.



### Автоматическая проверка
Система сопоставляет выбранные требования (например, наличие чекбокса «Отчет») с реальным наличием файлов в указанной папке, изменяя статус работы в реальном времени.
